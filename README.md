# csDAM
csDAM: Cosine Similarity-based Discriminative Analysis of Mixtures

## Quick Start
You can install the latest version of csDAM from GitHub by
```{r, eval = FALSE}
devtools::install_github("niccolodpdu/csDAM")
```
